Check BSP Version
#################

You can use the following command to check BSP and L4T version on **ROScube**.

1. Check L4T version.
---------------------

Please use the following command.

.. code-block:: bash

    cat /etc/nv_tegra_release


2. Check kernel version.
------------------------

Please use the following command.

.. code-block:: bash

    uname -a
