Setting System Configuration
############################

| Power on the **ROScube**.  
| Then follow steps to set the system configuration on **ROScube**.  

  * Language
  * Keyboard Layout
  * Location
  * Username and Password
  * Nvpmodel Mode 

After configuration, you'll see the Ubuntu desktop.
