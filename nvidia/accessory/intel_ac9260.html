Using Intel AC9260 WiFi CardÂ¶

The tutorial will guide you how to use Intel AC9260 on ROScube.

  1. Install the WiFi Card into ROScube

  2. Install add-apt-repository

sudo apt install software-properties-common

  1. Install Canonical backport driver

sudo add-apt-repository ppa:canonical-hwe-team/backport-iwlwifi
sudo apt-get update
sudo apt install backport-iwlwifi-dkms

  1. Reboot the system and you can use Intel AC9260 now.
